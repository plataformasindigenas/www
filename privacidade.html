Política de Privacidade

Como tratamos seus dados pessoais em conformidade com a LGPD.

Dados Coletados

Este site coleta dados pessoais exclusivamente por meio do formulário de contato, processado pelo serviço formsubmit.co. Os dados coletados são:

  • Nome
  • E-mail
  • Organização
  • Mensagem

Finalidade do Uso

Os dados coletados são utilizados exclusivamente para responder a consultas e solicitações de contato. Não compartilhamos, vendemos ou utilizamos seus dados para qualquer outra finalidade.

Cookies e Rastreamento

Este site é estático e não utiliza cookies, rastreadores ou ferramentas de análise de terceiros. Nenhum dado de navegação é coletado ou armazenado.

Serviços Externos

Este site carrega fontes tipográficas dos servidores do Google Fonts. Ao acessar o site, seu navegador pode enviar dados ao Google conforme a política de privacidade do Google.

Seus Direitos (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei n. 13.709/2018), você tem direito a:

  • Solicitar acesso aos seus dados pessoais
  • Solicitar a correção de dados incompletos ou desatualizados
  • Solicitar a exclusão dos seus dados pessoais
  • Revogar o consentimento para o uso dos seus dados

Exclusão de Dados

Para solicitar a exclusão dos seus dados pessoais, entre em contato pelo e-mail info@plataformasindigenas.org.
